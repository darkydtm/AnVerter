# Anverter Implementation Plan

> **For agentic workers:** Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Android-приложение в стиле MiUIX (HyperOS) для конвертации фиата и крипты со встроенным калькулятором.

**Architecture:** Single-Activity Jetpack Compose. UI на библиотеке `miuix`. Чистая логика конвертации и калькулятора вынесена в тестируемые классы без зависимостей от Android. Курсы тянутся из CoinGecko `/exchange_rates` (один запрос → фиат+крипта, привязка к BTC) только в foreground при наличии сети; никакой фоновой работы (нет WorkManager/Service/Receiver). Последние курсы кэшируются в DataStore для offline.

**Tech Stack:** Kotlin 2.0.21, Jetpack Compose (BOM), miuix 0.8.8, OkHttp, kotlinx.serialization, DataStore Preferences, Coroutines/Flow, ручной DI (AppContainer), JUnit.

## Global Constraints

- Лицензия: **GPL-3.0** (файл `LICENSE`, заголовки по необходимости).
- Сборка **только** через GitHub Actions (debug + release), оптимизированно и асинхронно (matrix, Gradle build/configuration cache, dependency cache, concurrency-cancel).
- `minSdk = 31`, `targetSdk = 35`, `compileSdk = 35`.
- Package: `com.anverter.app`, имя приложения: **Anverter**.
- Тема: Material You (светлая/тёмная + динамические цвета Android 12+), мост в `MiuixTheme`.
- **Запрет фоновой работы**: только `INTERNET` и `ACCESS_NETWORK_STATE` permissions; в манифесте один Activity; синхронизация курсов лишь когда приложение на переднем плане и сеть доступна.
- Git: каждая задача — отдельная ветка от `main`, микрокоммиты, merge `--no-ff`.
